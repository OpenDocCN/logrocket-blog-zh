# 前端开发预测

> 原文：<https://blog.logrocket.com/predictions-for-frontend-development/>

网络世界正在向前端移动。这意味着更多的技术、更多的复杂性和更多需要跟踪的东西。我们录制了一场网络研讨会，回顾了前端开发的一些最大趋势，包括:

*   以打字打的文件
*   高性能网络
*   GraphQL(图形 SQL)
*   反应堆
*   开发人员生产力
